### `▍` No momento

Engenheiro de Software Sênior na **[Pismo](https://pismo.io) / Visa** — fintech de
infraestrutura de pagamentos. Trabalho com plataformas financeiras de alta
performance: arquiteturas distribuídas com pico de **6.000 TPS** e mais de
**200 milhões de registros/dia**, em stack reativa na AWS com pipelines de dados
em Apache Spark e Databricks.

> Passei pelo suporte técnico antes do desenvolvimento. Entender como um sistema
> falha na prática mudou como eu o projeto — e isso ficou comigo.
